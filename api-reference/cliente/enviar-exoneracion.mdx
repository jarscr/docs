---
title: "Factura con Exoneración"
description: "Emitir una factura electrónica con exoneración de impuestos"
---

Envía una factura electrónica (tipoDoc `"1"`) con exoneración de impuestos. La exoneración se aplica dentro del objeto `impuesto` de cada línea del detalle de servicio.

<Note>
  Existen dos modalidades de exoneración: **autorizaciones** (emitidas por entidades autorizadas) y **gobierno** (instituciones públicas con artículo e inciso de ley).
</Note>

## Request

```
POST https://{empresa}.mipos.co.cr/api/enviar/items
```

### Campos principales

Los campos principales son idénticos a la [Factura Electrónica](/api-reference/cliente/enviar-factura): `emisor`, `numeroReferencia`, `codigoActividad`, `tipoDoc` (`"1"`), `receptor`, `folioType`, `fechaEmision`, `resumenFactura`, `otrosCargos`.

### Exoneración en el impuesto

La diferencia está dentro de cada elemento del array `impuesto` en `detalleServicio`, que incluye un objeto `exoneracion`:

<ParamField body="exoneracion" type="object" required>
  Datos del documento de exoneración.

  <Expandable title="Propiedades">
    <ParamField body="tipoDocumento" type="string" required>
      Tipo de documento de exoneración:
      - `"01"` — Compras autorizadas
      - `"02"` — Ventas exentas a diplomáticos
      - `"03"` — Autorizado por ley especial
      - `"04"` — Exoneraciones DGTH
      - `"05"` — Zonas Francas
      - `"99"` — Otros
    </ParamField>
    <ParamField body="numeroDocumento" type="string" required>
      Número del documento de exoneración (ej: `"AL-00080875-23"`).
    </ParamField>
    <ParamField body="nombreInstitucion" type="string" required>
      Código o nombre de la institución que emite la exoneración.
      - `"01"` — Instituciones del Estado
      - `"04"` — DGTH (Dirección General de Tributación y Hacienda)
    </ParamField>
    <ParamField body="fechaEmision" type="string" required>
      Fecha de emisión del documento de exoneración en formato ISO 8601.
    </ParamField>
    <ParamField body="porcentajeExoneracion" type="number" required>
      Porcentaje de exoneración (ej: `13` para exoneración total del IVA).
    </ParamField>
    <ParamField body="nombreInstitucionOtros" type="string">
      Nombre completo de la institución (requerido para tipo `"01"` gobierno).
    </ParamField>
    <ParamField body="articulo" type="string">
      Artículo de ley que fundamenta la exoneración (para instituciones gobierno).
    </ParamField>
    <ParamField body="inciso" type="number">
      Inciso del artículo de ley (para instituciones gobierno).
    </ParamField>
  </Expandable>
</ParamField>

## Ejemplo: Exoneración gobierno (con artículo e inciso)

Para instituciones del Estado, se agregan los campos `nombreInstitucionOtros`, `articulo` e `inciso`:

```json
"exoneracion": {
  "numeroDocumento": "7293",
  "fechaEmision": "1992-04-03T00:00:00",
  "nombreInstitucion": "01",
  "nombreInstitucionOtros": "CAJA COSTARRICENSE DEL SEGURO SOCIAL",
  "porcentajeExoneracion": 13,
  "tipoDocumento": "03",
  "articulo": "2",
  "inciso": 1
}
```
