---
title: "Consultar Exoneración"
description: "Validar un documento de exoneración en Hacienda"
---

Consulta y valida un documento de exoneración en el Ministerio de Hacienda. Útil para verificar que un número de exoneración es válido antes de incluirlo en una factura.

## Request

```
POST https://{empresa}.mipos.co.cr/api/hacienda/exoneracion
```

<ParamField body="exoneracion" type="string" required>
  Número del documento de exoneración a consultar (ej: `"AL-00080875-23"`).
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://{empresa}.mipos.co.cr/api/hacienda/exoneracion \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer {token}" \
  -d '{
    "exoneracion": "AL-00080875-23"
  }'
```

```php PHP
$ch = curl_init('https://{empresa}.mipos.co.cr/api/hacienda/exoneracion');
curl_setopt_array($ch, [
    CURLOPT_POST           => true,
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_HTTPHEADER     => [
        'Content-Type: application/json',
        'Authorization: Bearer ' . $token,
    ],
    CURLOPT_POSTFIELDS     => json_encode([
        'exoneracion' => 'AL-00080875-23',
    ]),
]);

$response = curl_exec($ch);
curl_close($ch);

$result = json_decode($response, true);
```

```javascript Node.js
const response = await fetch('https://{empresa}.mipos.co.cr/api/hacienda/exoneracion', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}`,
  },
  body: JSON.stringify({ exoneracion: 'AL-00080875-23' }),
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://{empresa}.mipos.co.cr/api/hacienda/exoneracion',
    headers={'Authorization': f'Bearer {token}'},
    json={'exoneracion': 'AL-00080875-23'},
)

data = response.json()
```
</RequestExample>
