---
title: "Registrar Emisor"
description: "Registrar un nuevo emisor con su certificado de Hacienda"
---

Registra un nuevo emisor en el sistema con sus credenciales del Ministerio de Hacienda y certificado digital (archivo `.p12` codificado en base64).

<Warning>
  El campo `certificadoHacienda` debe ser el archivo `.p12` del emisor codificado en base64. Este certificado es emitido por el BCCR (Banco Central de Costa Rica).
</Warning>

## Request

```
POST https://{empresa}.mipos.co.cr/api/enviar/emisor
```

### Datos del emisor

<ParamField body="identificacion" type="number" required>
  Número de identificación del emisor (cédula física o jurídica).
</ParamField>

<ParamField body="tipoIdentificacion" type="number" required>
  Tipo de identificación:
  - `1` — Cédula Física
  - `2` — Cédula Jurídica
  - `3` — DIMEX
  - `4` — NITE
</ParamField>

<ParamField body="codigoActividad" type="number" required>
  Código de actividad económica registrado en Hacienda (ej: `4772.0`).
</ParamField>

<ParamField body="sucursal" type="number" required>
  Número de sucursal. Usar `1` para la sede principal.
</ParamField>

<ParamField body="nombreComercial" type="string" required>
  Nombre comercial o razón social del emisor.
</ParamField>

<ParamField body="telefono" type="string" required>
  Número de teléfono del emisor.
</ParamField>

<ParamField body="correoElectronico" type="string" required>
  Correo electrónico del emisor.
</ParamField>

### Ubicación

<ParamField body="ubicacion" type="object" required>
  Dirección física del emisor.

  <Expandable title="Propiedades">
    <ParamField body="provincia" type="string" required>
      Código de provincia (1-7).
    </ParamField>
    <ParamField body="canton" type="string" required>
      Código de cantón (2 dígitos).
    </ParamField>
    <ParamField body="distrito" type="string" required>
      Código de distrito (2 dígitos).
    </ParamField>
    <ParamField body="barrio" type="string" required>
      Código de barrio (2 dígitos).
    </ParamField>
    <ParamField body="direccion" type="string" required>
      Dirección detallada en texto libre.
    </ParamField>
  </Expandable>
</ParamField>

### Credenciales de Hacienda

<ParamField body="ambiente" type="number" required>
  Ambiente de Hacienda:
  - `1` — Producción
  - `2` — Pruebas (Sandbox)
</ParamField>

<ParamField body="usuarioHacienda" type="string" required>
  Usuario de acceso al sistema de comprobantes electrónicos de Hacienda.
</ParamField>

<ParamField body="claveHacienda" type="string" required>
  Clave de acceso al sistema de comprobantes electrónicos (codificada en base64).
</ParamField>

<ParamField body="pinCertificado" type="string" required>
  PIN del certificado digital `.p12`.
</ParamField>

<ParamField body="certificadoHacienda" type="string" required>
  Certificado digital `.p12` codificado en base64.
</ParamField>
