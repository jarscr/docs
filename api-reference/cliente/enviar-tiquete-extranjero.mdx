---
title: "Tiquete Extranjero"
description: "Emitir un tiquete electrónico para receptor con identificación extranjera"
---

Envía un tiquete electrónico (tipoDoc `"4"`) para un receptor con identificación extranjera (tipo `"05"`).

<Note>
  La diferencia con el tiquete regular es que este incluye un receptor con tipo de identificación `"05"` (extranjero), que acepta números de pasaporte u otros documentos internacionales.
</Note>

## Request

```
POST https://{empresa}.mipos.co.cr/api/enviar/items
```

### Campos principales

<ParamField body="emisor" type="string" required>
  Número de identificación del emisor registrado.
</ParamField>

<ParamField body="numeroReferencia" type="integer" required>
  Número consecutivo de referencia del documento.
</ParamField>

<ParamField body="codigoActividad" type="string" required>
  Código de actividad económica del emisor.
</ParamField>

<ParamField body="tipoDoc" type="string" required>
  Tipo de documento. Para tiquete: `"4"`.
</ParamField>

<ParamField body="folioType" type="string" required>
  Tipo de folio. Siempre `"INVOICE"`.
</ParamField>

<ParamField body="fechaEmision" type="string" required>
  Fecha y hora de emisión en formato ISO 8601.
</ParamField>

### Receptor extranjero

<ParamField body="receptor" type="object" required>
  Datos del receptor extranjero.

  <Expandable title="Propiedades">
    <ParamField body="nombre" type="string" required>
      Nombre completo del receptor.
    </ParamField>
    <ParamField body="telefono" type="string">
      Número de teléfono.
    </ParamField>
    <ParamField body="identificacion" type="object" required>
      <Expandable title="Propiedades">
        <ParamField body="tipo" type="string" required>
          Tipo de identificación. Para extranjeros: `"05"`.
        </ParamField>
        <ParamField body="numero" type="string" required>
          Número de pasaporte o documento extranjero.
        </ParamField>
      </Expandable>
    </ParamField>
    <ParamField body="correoElectronico" type="string">
      Correo electrónico del receptor.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="detalleServicio" type="array" required>
  Líneas de detalle del documento. Ver [Factura Electrónica](/api-reference/cliente/enviar-factura) para la estructura completa.
</ParamField>

<ParamField body="otrosCargos" type="array">
  Cargos adicionales. Ver [Factura Electrónica](/api-reference/cliente/enviar-factura) para la estructura.
</ParamField>

<ParamField body="resumenFactura" type="object" required>
  Resumen con moneda y totales. Ver [Factura Electrónica](/api-reference/cliente/enviar-factura) para la estructura.
</ParamField>
